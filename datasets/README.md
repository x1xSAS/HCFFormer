# Dataset Directory

Place benchmark datasets in this directory using the following structure:

```text
datasets/
|
|-- Texas/
|   |-- A/
|   |-- B/
|   |-- label/
|
|-- California/
|   |-- A/
|   |-- B/
|   |-- label/
|
|-- Shuguang/
    |-- A/
    |-- B/
    |-- label/
```

Dataset files are not included in this repository due to licensing and distribution restrictions. Users should obtain the datasets from their original sources as described in the associated paper, and organize them according to the directory structure shown below.

Recommended split ratio:

- Training: 70%
- Validation: 10%
- Testing: 20%
