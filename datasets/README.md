# Dataset Directory

Place benchmark datasets in this directory using the following structure:

```text
datasets/
|
|-- Texas/
|   |-- A/
|   |-- B/
|   |-- label/
|
|-- California/
|   |-- A/
|   |-- B/
|   |-- label/
|
|-- Shuguang/
    |-- A/
    |-- B/
    |-- label/
```

Dataset files are not included in this repository unless redistribution permission is available.

Recommended split ratio:

- Training: 70%
- Validation: 10%
- Testing: 20%
